jquery.xdCheckbox Плагин

Страница проекта | Скачать исходники

пользовательская стилизация элементов checkbox при помощи jquery плагина

ru | eng

Так выглядит checkbox без xdCheckbox

Добавим в код страницы

<script type="text/javascript" src="jquery.min.js"></script>
<script type="text/javascript" src="jquery.xdcheckbox.js"></script>
<link rel="stylesheet" type="text/css" href="jquery.xdcheckbox.css"/>

Для стилизации checkbox используется такая конструкция

$(function(){
	$('input[type=checkbox]').xdCheckbox();
})


Disabled
Not checked

JavaScript API изменения checked и disabled

$(function(){
	$('input').xdCheckbox();
	$('input:first').xdCheckbox({disabled:true,checked:true});
})


Свои размеры

$(function(){
	$('input').xdCheckbox({width:32});
})
